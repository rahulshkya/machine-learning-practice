# ModernShop - Modern E-Commerce Website 🛍️

A beautiful, animated e-commerce website built with React and pure CSS animations. No external animation libraries needed!

## Features ✨

- **Smooth Animations**: Modern CSS animations for all interactions
- **Product Catalog**: Browse 12+ products across multiple categories
- **Shopping Cart**: Add/remove items with quantity management
- **Search & Filter**: Search products and filter by category
- **Newsletter**: Subscribe to get exclusive offers
- **Responsive Design**: Works perfectly on all devices
- **Modern UI**: Gradient backgrounds and sleek card designs

## Tech Stack 🚀

- **React 18**
- **Vite** (Lightning-fast development)
- **Pure CSS3** (No animation libraries)
- **React Hooks** (State management)

## Installation 📦

```bash
# Navigate to the project
cd ecommerce-store

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Key Features Explained 🎨

### Animations Include:
- **Fade In/Up**: Smooth entrance effects
- **Hover Effects**: Cards lift on hover
- **Floating Objects**: Smooth floating animations
- **Pop Animation**: Badge updates with visual feedback
- **Slide Animations**: Navigation and transitions
- **Scale Transforms**: Interactive button responses

### Components:
- **Navbar**: Search, categories, cart icon with badge
- **Hero Section**: Eye-catching landing area
- **Product Cards**: With hover effects and quick add-to-cart
- **Shopping Cart**: Full cart management with totals
- **Newsletter**: Email subscription form
- **Responsive Design**: Mobile-friendly layout
